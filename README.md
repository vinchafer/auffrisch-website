# Auffrisch.ch

Website für den Schweizer KMU Web-Refresh Service.

## Stack
- Einzelne HTML-Datei (index.html) — kein Framework, kein Build-Step
- Fonts: Playfair Display + DM Sans (Google Fonts)
- Formspree für Kontaktformular

## Setup
1. Formspree ID eintragen: in index.html nach `XXXXXXXX` suchen und ersetzen
2. Vercel: einfach den Ordner deployen, index.html wird automatisch erkannt
3. Domain: auffrisch.ch in Vercel verbinden
